# GUIA

## Requirements

	- nodejs
	- mongodb

## Installation
    git clone git://github.com/kersten/vdr-guia.git ./GUIA
    cd GUIA
    npm update

## Configuration

To configure GUIA you have to edit the file _src/etc/config_ and change
the parameters to suite your needs.
